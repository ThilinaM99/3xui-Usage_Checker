## 🚀 Requirements & Installation
1. Make sure to take a backup of the system before doing anything.
2. TMS-VPN only supports 3X-UI v2.8.4 or later.
3. Run the following command on your 3x-ui server to install or update:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/ThilinaM99/3xui-Usage_Checker/main/install.sh)
```

## 🧹 Uninstallation
If you wish to return to the Stock Version, run:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/ThilinaM99/3xui-Usage_Checker/main/uninstall.sh)
```
> [!WARNING]
> This will update your x-ui panel to the latest version(no data loss)
>
